# Razorpay Spotify Internship Project

## Description
This project is developed as part of an internship.
It includes Razorpay payment gateway integration
and Spotify API usage for music-related features.

## Features
- Razorpay payment integration
- Spotify API integration
- Secure transaction flow
- Internship project structure

## Technologies Used
- HTML
- CSS
- JavaScript
- Razorpay
- Spotify API

## How to Run
1. Clone or download the repository
2. Open the project folder
3. Add Razorpay and Spotify API keys
4. Run on browser or local server
